# YUVA Expense Tracker 💰
### **வயதானவர்களுக்கான எளிய தமிழ் குரல் வழி செலவு கணக்கு செயலி**
*(An AI-Based Tamil Voice Expense & Savings Tracker for Elderly People)*

---

## 🌟 Overview
**YUVA Expense Tracker** is a specialized financial management application designed with simplicity and accessibility in mind. Recognizing the challenges elderly people face with complex interfaces, YUVA offers a **voice-first** experience in Tamil, allowing users to record their daily expenses naturally by just speaking.

---

## ✨ Key Features

### 🎙️ AI-Powered Tamil Voice Input
*   **Natural Speech Recognition**: No need to type. Just say *"500 ரூபாய் மளிகை"* (500 rupees groceries) or *"ஆயிரம் ரூபாய் கரண்ட் பில்"* (1000 rupees current bill).
*   **Smart Parsing**: Automatically extracts the amount and category from the spoken Tamil sentence.

### 📊 Comprehensive Dashboard
*   **Instant Overview**: Real-time display of Monthly Income (சம்பளம்), Total Expenses (மொத்த செலவு), and remaining Savings (சேமிப்பு).
*   **Quick Categories**: Fast-access icons for common expenses like Medicine, Groceries, Bills, and Transport.

### 📈 Visual Analytics (Summary Screen)
*   **Dynamic Pie Charts**: View your spending distribution across different categories.
*   **Month-by-Month Filter**: Easily toggle between months and years to track historical spending.

### 📄 Professional PDF Reports
*   **One-Tap Export**: Generate detailed expense reports in PDF format.
*   **Perfect Tamil Formation**: Advanced native rendering ensures Tamil words are perfectly formed without broken characters in the PDF.

### 📝 History Management
*   **Detailed Log**: A chronological list of all your spending.
*   **Edit & Delete**: Full control to modify or remove incorrect entries.

### 👤 Profile & Privacy
*   **Custom Avatars**: Personalize the app with your own photo.
*   **Local Storage**: Your financial data is stored securely on your device using Hive. No data ever leaves your phone.

## 🛠️ Technology Stack
*   **Framework**: [Flutter](https://flutter.dev) (v3.x)
*   **Language**: Dart
*   **State Management**: Provider
*   **Database**: Hive (Local NoSQL)
*   **Speech Recognition**: Speech-to-Text
*   **UI/UX**: Material 3 with Noto Sans Tamil fonts

## 🚀 Installation

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/Guru-CodesAI/Yuva-Expense-Tracker.git
    ```
2.  **Install Dependencies**:
    ```bash
    flutter pub get
    ```
3.  **Run the App**:
    ```bash
    flutter run
    ```
4.  **Download the APK**:
    Go to the [Releases](https://github.com/Guru-CodesAI/Yuva-Expense-Tracker/releases) page to download the latest `app-release.apk`.
